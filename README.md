# Algorand Project – Digital Asset Tracker

## Project Description
This project is built on the Algorand blockchain to track digital assets securely and transparently.

## Objectives
- Learn Algorand blockchain development
- Create smart contracts
- Manage digital assets
- Build decentralized application

## Tools & Technologies
- Algorand
- Python / JavaScript
- Algo SDK
- GitHub

## Project Plan
1. Setup Algorand environment
2. Create Algorand account
3. Create smart contract
4. Asset creation and transfer
5. Build simple frontend
